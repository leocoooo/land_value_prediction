# Dossier raw

Ce dossier contient les **données brutes originales**, téléchargées avant tout traitement ou nettoyage.

## Contenu typique :
- Fichiers DVF (`ValeursFoncieres-20XX.txt`)
- Données d’accessibilité (`accessibilite-en-gare.csv`, `metro-france.csv`)
- Données géographiques (`centroides-communes-population.geojson`)
- Indicateurs macroéconomiques (`Webstat_Export_fr_ICP...csv`)
