# Dossier processed

Ce dossier contient les fichiers de données **nettoyés et enrichis** utilisés pour la modélisation du prix au mètre carré en Île-de-France.

- `idf_immobilier_full.csv` : données agrégées par trimestre et commune  
- `idf_immobilier_enriched.csv` : données enrichies avec les coordonnées, gares et métros  
- `idf_immobilier_filled_arrondissements.csv` : version finale sans doublons (hors 75056)
